随机指标(STOCH)

# 随机指标（Stochastic Oscillator）

# 历史

随机震荡指标 (STOCH) 由George Lane在1950年代开发。Lane认为他的指标是衡量动量的好方法，这很重要，因为动量的变化先于价格的变化。在 2007 年的一次采访中，他被引述说“随机指标衡量价格的动量。如果你想象一枚火箭在空中升起 — 在它能够下降之前，它必须减速。动量总是在价格之前改变方向。”

# 计算

随机指标通常由三条线组成：

  1. %K 线：表示当前收盘价相对于最近一段时间内价格范围的位置。
  2. %D 线：是 %K 线的平滑线，通常采用 %K 线的移动平均值。
  3. 中间线：一条固定的水平线，通常标记为 20 和 80，用于判断超买和超卖状态。

随机指标的计算步骤如下：

  1. 计算最近一段时间内的最高价（H）和最低价（L）。
  2. 计算当前收盘价（C）相对于最高价和最低价的位置（%K 值）： %K = (C - L) / (H - L) * 100
  3. 计算 %K 值的移动平均（%D 值）： %D = 移动平均(%K, N) 其中，N 为平均的期数，一般取 3、5、10 等。

随机指标的数值范围在 0 到 100 之间，常用的判断方法是当 %K 值超过 80 时，表示超买状态，建议卖出；当 %K 值低于 20 时，表示超卖状态，建议买入。

请注意，这只是随机指标的基本概念和计算方法，实际使用时可能会有不同的变体和参数配置。

# 基础

随机震荡指标是一个区间震荡指标，由在 0 和 100 之间移动的两条线组成。第一条线（称为 %K）显示当前收盘价相对于用户定义周期的高/低范围。第二条线（称为 %D）是 %K 线的简单移动平均线。现在，与大多数指标一样，随机指标中使用的所有周期都可以由用户定义。也就是说，最常见的选择是 14 周期的 %K 和 3 周期的 %D SMA。

基本理解是随机指标使用收盘价来确定动量。 当价格收于回顾期高/低范围的上半部分时，随机震荡指标 (%K) 上升也表明动能或买/卖压力增加。 当价格收于该期间高/低范围的下半部分时，%K 下降，表明势头减弱或买/卖压力。

# 要找什么

超买/超卖

与任何区间指标非常相似，超买/超卖情况是由随机震荡指标产生的主要信号。超卖的默认阈值为 20，超买的默认阈值为 80。这些是典型水平，但可能并不适合所有情况，具体取决于所交易的金融商品。通过一些实验和历史分析来找到正确的水平。 请记住，在使用随机指标识别超买/超卖水平时，通常最好顺应趋势进行交易。原因是超买并不总是意味着看跌，就像超卖并不总是意味着看涨。很多时候，超买（超卖）情况可能是趋势加强的迹象，不一定是即将发生的逆转。

# 概括

随机震荡指标是识别动量变化的经典工具。 它是一种多功能指标，可用于各种时间周期（天、周、月、日内），这增加了它的受欢迎程度。在生成信号方面，随机震荡指标确实可以生成高质量的信号。但请记住，当将其用作信号发生器时（尤其是对于背离和看涨/看跌设置），最好顺应趋势使用。 技术分析师应该了解市场的整体趋势。将随机指标与趋势线等其他技术分析手段一起使用以确定市场方向并非不明智。

# 代码示例


/**
 * 计算 Stochastic 技术指标的值
 *
 * @param {Prices} prices - 价格数组对象，包括最高价、最低价和收盘价
 * @param {number} periodK - Stochastic %K 周期长度
 * @param {number} periodD - Stochastic %D 周期长度
 * @returns {StochasticResult} - 计算得到的 Stochastic 技术指标结果对象
 */

interface Prices {
  highs: number[];
  lows: number[];
  closes: number[];
}

interface StochasticResult {
  k: number;
  d: number;
  slowK: number;
  slowD: number;
}

function calculateStochastic(prices: Prices, periodK: number, periodD: number): StochasticResult {
  const { highs, lows, closes } = prices;

  if (highs.length !== lows.length || highs.length !== closes.length) {
    throw new Error('输入数据长度不一致');
  }

  const kValues = [];

  for (let i = periodK - 1; i < highs.length; i++) {
    const highestHigh = Math.max(...highs.slice(i - periodK + 1, i + 1));
    const lowestLow = Math.min(...lows.slice(i - periodK + 1, i + 1));
    const close = closes[i];

    const k = 100 * ((close - lowestLow) / (highestHigh - lowestLow));
    kValues.push(k);
  }

  const dValues = calculateSMA(kValues, periodD);

  const slowKValues = calculateSMA(kValues, periodD);
  const slowDValues = calculateSMA(dValues, periodD);

  return {
    k: kValues[kValues.length - 1],
    d: dValues[dValues.length - 1],
    slowK: slowKValues[slowKValues.length - 1],
    slowD: slowDValues[slowDValues.length - 1]
  };
}

# 判断

/**
 * 根据 Stochastic 技术指标计算得到的 K、D、SlowK 和 SlowD 值，判断买入、卖出或中立信号。
 * 当K值小于20、大于D值、大于SlowK值和D值大于SlowD值时，判断为买入信号；当K值大于80、小于D值、小于SlowK值和D值小于SlowD值时，判断为卖出信号；否则判断为中立信号
 *
 * @param {number} k - Stochastic 技术指标计算得到的 K 值
 * @param {number} d - Stochastic 技术指标计算得到的 D 值
 * @param {number} slowK - Stochastic 技术指标计算得到的 SlowK 值
 * @param {number} slowD - Stochastic 技术指标计算得到的 SlowD 值
 * @returns {string} - 根据 K、D、SlowK 和 SlowD 值判断出的买入、卖出或中立信号
 */
export function getStochasticSignal(k: number, d: number, slowK: number, slowD: number): string {
  if (k < 20 && k > d && k > slowK && d > slowD) {
    return "BUY";
  } else if (k > 80 && k < d && k < slowK && d < slowD) {
    return "SELL";
  } else {
    return "NEUTRAL";
  }
}
