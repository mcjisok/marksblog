# 什么是移动平均线

移动平均线是一种常用的技术分析工具，用于平滑价格数据并显示价格的趋势。它通过计算一定时间内的平均价格来消除价格波动的短期噪音，从而更清晰地展示价格的走势。

移动平均线的计算方式很简单，它基于一段时间内的价格数据，计算出每个时间点的平均值。常用的移动平均线包括简单移动平均线（Simple Moving Average，SMA）和指数移动平均线（Exponential Moving Average，EMA）。

  1. 简单移动平均线（SMA）：简单移动平均线是最基本的移动平均线类型。它使用相同权重的价格数据计算平均值。计算公式为：

    SMA = (P1 + P2 + P3 + ... + Pn) / n

    其中，P1到Pn是一段时间内的价格数据，n是选定的时间周期。

    例如，如果要计算5日简单移动平均线，就将最近5个交易日的收盘价相加，然后除以5。

  2. 指数移动平均线（EMA）：指数移动平均线是一种赋予较高权重给最近价格数据的移动平均线。它更加关注近期价格的变动，对较早期的价格数据给予较低的权重。计算公式为：

    EMA = (P * K) + (EMA_previous * (1 - K))

    其中，P是当前的价格，EMA_previous是前一个周期的指数移动平均线值，K是平滑因子，它的计算公式为 2 / (n + 1)，n是选定的时间周期。

    与简单移动平均线相比，指数移动平均线对近期价格变动更加敏感。

移动平均线可以用于识别价格的趋势、判断支撑和阻力水平、确认买卖信号等。常用的周期包括10日、20日、50日和200日移动平均线，不同的周期可以反映不同的时间尺度。

移动平均线可以在股票、期货、外汇等金融市场的技术分析中广泛应用，它是许多交易策略和指标的基础组成部分。

function calculateMovingAverages(closePrices: number[], days: number[] = [5, 10, 20]): Record<string, number[]> {
  const movingAverages: Record<string, number[]> = {};

  days.forEach((day: number) => {
    const sma: number[] = [];
    const ema: number[] = [];
    // 计算简单移动平均线（SMA）
    for (let i = day - 1; i < closePrices.length; i++) {
      const sum = closePrices.slice(i - day + 1, i + 1).reduce((acc, price) => acc + price, 0);
      const average = sum / day;
      sma.push(average);
    }

    // 计算指数移动平均线（EMA）
    let emaPrevious = closePrices.slice(0, day).reduce((acc, price) => acc + price, 0) / day;
    ema.push(emaPrevious);
    const k = 2 / (day + 1);
    for (let i = day; i < closePrices.length; i++) {
      const emaCurrent = (closePrices[i] * k) + (emaPrevious * (1 - k));
      ema.push(emaCurrent);
      emaPrevious = emaCurrent;
    }
    movingAverages[`sma${day}`] = sma;
    movingAverages[`ema${day}`] = ema;
  })

  return movingAverages;
}
